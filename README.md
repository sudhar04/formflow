# FormFlow

A modern and responsive file upload and form management application built with **React**, **TypeScript**, **Vite**, and **Tailwind CSS**. FormFlow provides a clean, intuitive interface for collecting user information and uploading files securely.

---

## 🌐 Live Demo

🔗 https://flowstate-form.lovable.app

---

## ✨ Features

- Modern and responsive UI
- Drag-and-drop file upload
- Secure form submission
- Mobile-friendly design
- Fast performance with Vite
- Reusable React components
- Clean and scalable project structure

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/sudhar04/formflow.git
```

### Navigate to Project

```bash
cd formflow
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

---

## 📦 Build

npm run build
